# Assets Folder

This folder should contain the following image files for the app:

- `icon.png` - App icon (1024x1024px)
- `splash.png` - Splash screen image
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px)

For development purposes, you can use placeholder images or generate them using online tools.

The app will work without these assets, but you may see warnings in the console.
